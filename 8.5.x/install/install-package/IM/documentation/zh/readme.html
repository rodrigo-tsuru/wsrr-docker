  • 问题发生时您运行的是什么软件版本？
  • 您具有与问题相关的日志、跟踪或消息吗？
  • 您可以重现该问题吗？如果可以，那么重现该问题需要哪些步骤？
  • 该问题存在变通方法吗？
  • 如果存在，请准备描述变通方法。
