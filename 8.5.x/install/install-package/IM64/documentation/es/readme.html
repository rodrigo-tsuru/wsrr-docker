Soporte de software de IBM Rational

Para obtener la información de contacto y las directrices o los materiales de consulta que necesitará cuando necesite ayuda, lea el documento IBM Software Support Handbook.

Para conocer las preguntas frecuentes (FAQ), las listas de problemas conocidos y sus arreglos y otra información de soporte, visite el portal de soporte de IBM Installation Manager.

Para conocer las noticias y los eventos del producto, así como otras informaciones, visite el sitio web de IBM Rational Software.

Antes de ponerse en contacto con IBM Rational Software Support, recopile la información de los antecedentes que necesita para describir su problema. Cuando describa un problema a un especialista de soporte de software de IBM, sea tan específico como sea posible e incluya toda la información de antecedentes relevantes para que el especialista pueda ayudarle a resolver el problema de forma eficiente. Para ahorrar tiempo, debe conocer las respuestas a estas preguntas:

  • ¿Qué versiones de software estaba ejecutando cuando se produjo el problema?
  • ¿Tiene anotaciones, rastreos o mensajes relacionados con el problema?
  • ¿Puede reproducir el problema? ¿Si es así, qué pasos sigue para reproducirlo?
  • ¿Hay un método alternativo para evitar el problema?
  • Si es así, esté preparado para describir el método alternativo.
